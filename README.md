### SCSS

작성순서는 html을 기준으로 하여 탑다운 방식이며,
id값은 따로 빼서 적용.
초기화 간단하게 적용 (추후 추가예정)
폰트 적용 완료 (Tahoma, 영문전용 폰트)

마일리지 이미지들을 전부 opacity: 0, 클래스로 묶은다음 nth-child를 통해서 배경이미지로 삽입. 위치또한 :nth-child를 통해서 세로와 가로 적용함.
